Dimensions Networks Generator: Topics

This website was automatically generated by dimensions-network-gen, a prototype Python application that allows to boostrap the construction of network visualizations using the Dimensions COVID-19 dataset and VOSViewer.

How it works. Network visualizations can be triggered by passing a single SQL query returning publication Dimension IDs to the dimensions-network-gen application. Each query represents a topic, i.e. a subset of the Dimensions dataset. The application will then automatically perform a network analysis on the dataset and output a visualization using VOSViewer.

For more information, see the STI-2022 paper and the README on GitHub.
