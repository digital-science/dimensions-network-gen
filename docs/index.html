COVID-19 Scientific Networks

This website was created using dimensions-network-gen, an open-source Python application that allows to construct network visualizations using the Dimensions COVID-19 dataset and VosViewer.

The topics listed below correspond to subsets of the Dimensions COVID-19 dataset. Clicking on a topic allows to explore it using different interactive network visualizations.

New topics can be defined by passing a SQL query returning publication `ids` to the dimensions-network-gen application. The application will then generate the network and visualize it using VosViewer (more details).
